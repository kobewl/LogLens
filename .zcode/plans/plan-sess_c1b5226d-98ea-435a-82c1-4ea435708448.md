## Part 2: 云日志查询功能

### 2.1 后端：新增统一查询命令

**`src-tauri/src/commands.rs`** — 新增 `cloud_search_logs` 命令：

```rust
#[tauri::command]
async fn cloud_search_logs(
    provider: String,        // aliyun/tencent/huawei
    project_name: String,    // 项目名（从导入配置中获取凭据）
    alias_name: String,      // 别名（获取 logstore/stream_id）
    keywords: Option<String>,
    time_from: Option<String>,  // 毫秒时间戳
    time_to: Option<String>,
    limit: Option<i32>,
    state: State<'_, AppState>,
) -> Result<serde_json::Value, String>
```

逻辑：
- 从数据库读取 project credentials + alias 信息
- **阿里云**：调 MCP `query_logs` 工具，参数 projects/logstore/keywords/时间
- **腾讯云**：调 MCP `SearchLog` 工具，参数 TopicId/Query/时间
- **华为云**：调 REST API `huawei::query_logs`

### 2.2 前端：云日志查询面板

**`src/components/cloud/CloudSourcePanel.tsx`** — 别名列表扩展为可交互：

- 点击别名 → 展开查询表单（关键词输入框 + 时间范围 + 搜索按钮）
- 搜索结果显示在下方（pre 标签，可滚动）
- 加载态使用 Loader2 spinner
